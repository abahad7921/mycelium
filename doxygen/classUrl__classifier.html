Detailed Description

A classifier which queues up urls grouped by host in N queues.

top_q is queue for elements yet not classified

               __ queue # 1  host X
  top_q       /
  -----------/___ queue # 2  host Y
  host       \
  {!X,!Y,!Z}  \__ queue # 3  host Z
